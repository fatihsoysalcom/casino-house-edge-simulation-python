# Casino House Edge Simulation Python

This Python script simulates a simple coin flip casino game over many rounds. It demonstrates the concept of "house edge" by setting slightly unfavorable payout odds for the player (e.g., winning $1.95 for a $1 bet on a 50/50 chance, instead of $2). Over a large number of simulated games, the casino's balance consistently increases, illustrating how casinos ensure long-term profitability through statistical advantage, even with random outcomes.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open a terminal or command prompt.
3. Navigate to the directory where you saved the file.
4. Run the script using `python main.py`.
